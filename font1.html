HTML: ما هي؟

HTML هي اختصار لـ HyperText Markup Language أي "لغة ترميز النص التشعبي"، وهي اللغة الأساسية لبناء صفحات الويب.

✏️ البنية العامة لملف HTML

<!DOCTYPE html> <html> <head> <title>عنوان الصفحة</title> </head> <body> <h1>مرحبًا بك</h1> <p>هذه صفحة HTML بسيطة</p> </body> </html>
✅ ملاحظة: وسم <!DOCTYPE> يخبر المتصفح أنك تستخدم HTML5.

📌 العناصر الرئيسية

  • <html>: الجذر الأساسي للصفحة
  • <head>: يحتوي على بيانات وصفية
  • <body>: يحتوي على المحتوى المرئي
